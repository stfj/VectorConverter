mac os x gui for https://github.com/visioncortex/vtracer and https://github.com/upscayl/upscayl, and that also includes a bunch of editing and vector simplification tools.

paste or drag a bitmap in.
command-c or command-s to get a svg out.
space bar to pan / hide control points
z to zoom tool
v to cursor tool

delete / command-z to delete shapes you dont want or undo that.
